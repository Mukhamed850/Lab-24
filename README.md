# Лабораторная работа №4 (год 2)

## Библиотека Qt: сетевое взаимодействие

Разработана программа с использованием библиотеки Qt. В окне программы 
реализовано главное меню с пунктами "Файл" и "Помощь". В меню "Файл" добавлены
пункты "Запустить сервер" и "Остановить сервер", а также "Открыть изображение" 
и "Выход". 

При выборе "Открыть изображение" появляется диалоговое окно для открытия 
файла, который будет отправляться сервером в ответ на HTTP-запрос "GET /". 
При открытии изображения его содержимое отображается на 2/3 окна.

В меню "Помощь" добавлен пункт "О программе" выводящий диалоговое окно с 
информацией об авторе и версии библиотеки Qt с которой была собрана и запущена 
программа.

Для реализации сетевого взаимодействия задействованы возможности классов 
QTcpSocket и QTcpServer.

Проект создан на основе qmake.

Исходные коды программы снабжены краткими комментариями.

UML 2.0 диаграмма разработанных классов включена в состав исходных кодов 
программы в виде файла формата PDF.
